📄 System Architecture

Resume Screening and Career Guidance System
Slide 1/8

System Overview

A web-based application designed to automate resume screening for HR professionals and provide career guidance to job seekers.

Key Features

  • Resume upload and text extraction
  • Automatic skill identification
  • Resume scoring against job requirements
  • Candidate ranking and filtering
  • Skill gap analysis
  • Career roadmap generation

Target Users

  • HR Professionals: Upload multiple resumes, filter and rank candidates
  • Candidates: Get resume score, skill gaps, and career guidance

Architecture Pattern

Three-Tier Architecture with clear separation:

  • Presentation Layer (React.js)
  • Application Layer (Node.js/Express)
  • Processing Layer (Python/FastAPI)

📈 Scoring Algorithm

Match Score Calculation Method
Slide 7/8

Scoring Formula

Score = 0.45×Skills + 0.40×Semantic + 0.15×Keywords

Weight Distribution

ComponentWeightMethod
Skill Match45%Set intersection ratio
Semantic40%BERT cosine similarity
Keywords15%TF-IDF cosine similarity

Why This Weighting?

  • Skills (45%): Direct measurement of job fit
  • Semantic (40%): Captures context and meaning
  • Keywords (15%): Ensures terminology match
